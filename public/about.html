About the Project

Understanding Internet Exposure

SentinelScope is an educational cybersecurity tool built to help everyday users, IT teams, and students see exactly what information about their systems is publicly visible on the internet — and understand why that matters.

The Problem

Why Exposure Monitoring Matters

🌐
Hidden Attack Surfaces
Millions of internet-connected devices — routers, servers, cameras, databases — are inadvertently left exposed to the public internet without proper security configurations. Owners are often completely unaware.
🔍
Attackers Use Shodan Too
Tools like Shodan continuously crawl the internet and index everything they find. Malicious actors already use these indexes to locate vulnerable targets. SentinelScope puts the same visibility in the hands of defenders.
📊
Data Without Context Is Useless
Raw Shodan data is complex and technical. SentinelScope transforms it into readable dashboards, risk scores, and visual charts so non-experts can understand and act on the findings.
🛡️
Awareness Drives Better Security
Organizations that know about their exposed attack surface take action to secure it. SentinelScope aims to be the first step in that awareness — surfacing information so it can no longer be ignored.

Architecture

How SentinelScope Works

01
User Enters a Target
A domain name or IP address is entered into the search form on the home page. The target is passed as a URL parameter to the results page.
02
Backend Queries Shodan
The Express.js backend resolves domains to IPs via Shodan's DNS API, then performs a full host lookup to retrieve open ports, services, banner data, CVEs, and geolocation.
03
Results Are Rendered
The shaped data is sent to the frontend where Chart.js renders port and service visualizations and Leaflet.js plots the target location on an interactive map.
04
Scan Saved to Database
Each completed scan is written to a Supabase PostgreSQL database so the history can be retrieved and displayed on the home page for future reference.
