# SentinalScope: Cyber Exposure Monitoring Dashboard

## Project Description
SentinalScope is a cybersecurity exposure monitoring dashboard project that hopes to solve the issue of publicly exposed internet connected devices and services. This project uses the Shodan API to retrieve publicly available cybersecurity information associated with a domain name or IP address.

Users can:
- Search a domain name or IP address to view publicly explosed cybersecurity info
- Identiy open porta and internet-facing services
- View vulnerabilities that are linked to exposed systems
- Organization information
- Geographic location data
- Analyze risk levels
- Track scan history

The goal of this project is to provide a web application that can be used for educational purposes to reveal publicly exposed information associated with domains or IP addresses. By displaying information such as open ports, running services, vulnerabilities, and geographical data in an understandable format, users can have a better understanding of cybersecurity risk and the importance of securing internet-facing infrastructure. 

Hopefully with the help of SentinelScope, we can increase awareness about attack surfaces, and encourage stronger cybersecurity practices for the future.
